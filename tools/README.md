# 🔧 STS Tools & Utilities

This directory contains utility scripts and tools for working with the Sensory Tracer Science framework.

## 📁 **Available Tools**

### **🧪 Validation & Testing**
- `validate_implementation.py` - Comprehensive STS compliance validation
- `run_biocompatible_tests.py` - Biocompatible tracer testing suite
- `benchmark_performance.py` - Performance benchmarking and optimization
- `check_framework_integrity.py` - Framework consistency verification

### **📊 Analysis & Visualization**
- `plot_tracer_results.py` - Generate comprehensive result visualizations
- `analyze_toxicity_data.py` - Toxicity analysis and safety assessment
- `compare_implementations.py` - Compare different tracer implementations
- `generate_reports.py` - Automated report generation

### **🔧 Development Utilities**
- `setup_development.py` - Development environment setup
- `format_code.py` - Code formatting and style checking
- `update_documentation.py` - Documentation generation and updates
- `create_new_tracer.py` - Template generator for new tracer implementations

### **📚 Educational Tools**
- `interactive_demo.py` - Interactive STS demonstrations
- `tutorial_generator.py` - Generate step-by-step tutorials
- `example_creator.py` - Create educational examples
- `concept_visualizer.py` - Visualize STS concepts and principles

## 🚀 **Quick Usage**

### **Validation**
```bash
# Validate STS implementation
python tools/validate_implementation.py

# Test biocompatible tracers
python tools/run_biocompatible_tests.py

# Check framework integrity
python tools/check_framework_integrity.py
```

### **Analysis**
```bash
# Plot comprehensive results
python tools/plot_tracer_results.py --input results.json --output plots/

# Analyze toxicity data
python tools/analyze_toxicity_data.py --data toxicity_results.csv

# Generate automated report
python tools/generate_reports.py --experiment neural_tracer_study
```

### **Development**
```bash
# Setup development environment
python tools/setup_development.py

# Create new tracer template
python tools/create_new_tracer.py --name "MyNewTracer" --type "optical"

# Format code
python tools/format_code.py --directory sensory_tracer_science/
```
